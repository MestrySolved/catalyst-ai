# Project Catalyst: Agentic Cloud Automation

Project Catalyst is an **Autonomous Enterprise Migration Framework** powered by the Gemini Enterprise Agent Platform. It manages the lifecycle of 5 core namespaces across Dev and Prod environments in the GCP Mumbai region.

## 🚀 Capabilities
- **Autonomous Triage:** Analyzes infrastructure state via Gemini 1.5 Pro.
- **Self-Healing:** Monitors Dynatrace metrics to trigger proactive remediation.
- **Interactive Hub:** Uses JupyterLab as a governance portal for human-in-the-loop approvals.

## 🛠️ Integrated Stack
- **GitHub:** Source code and Agent logic.
- **GCP:** GKE Regional Clusters (asia-south1).
- **Codefresh:** Parameterized CI/CD orchestration.
- **Dynatrace:** Real-time observability and SLI tracking.
